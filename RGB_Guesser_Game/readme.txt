RGB Guesser game built using HTML + CSS + Javascript

How to Play:
- You have to guess the color that corresponds to the RGB color displayed at the top of the screen
- The game allows you to choose between easy and hard modes where you have to select between choosing from 3 or 6 colors respectively.
- Colors are generated randomly.
- Click on a color to select it. If it is correct, you get an option to play again.
- If it is wrong, you can keep on clicking on the squares until you get the correct answer.
- Click on "New Colors" at any time to choose a differnet set of colors

Hint: Additive color mixing: 
- adding red to green yields yellow; 
- adding red to blue yields magenta; 
- adding green to blue yields cyan; 
- adding all three primary colors together yields white
